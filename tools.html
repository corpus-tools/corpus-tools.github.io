Tools

Converters

For corpora to be imported into the system, they must be converted into the native relational database format relANNIS using the Pepper converter framework. The current version of Pepper can convert stand-off PAULA XML, EXMARaLDA, generic XML, TreeTagger output, Tiger XML, <tiger2/>, ELAN, RST Tool and MMAX2 into relANNIS directly, to name a few formats. Other formats, including PALinkA, WebAnno XMI and MS Excel may be converted to PAULA XML first and then to relAnnis. For the latest version, see the SaltNPepper homepage.

SaltNPepper: (homepage)

Other Converters:
  • XSLT to PAULA converters (Tiger XML / EXMARaLDA / MMAX2 / Generic XML - to PAULA XML; NB - these work under a native UNIX OS but may not work under Cygwin)
  • Online PAULA converters
  • Concatenation script in Perl to merge multiple relAnnis corpora (creates one big corpus containing input relANNIS corpora as subcorpora)
  • Excel Plugin (EXMARaLDA <> Excel and Excel to PAULA XML; Windows only)
Further documentation on the PAULA XML format may be found here.

Annotation Tools

ANNIS data is produced using a variety of annotation tools. Some frequently used tools include (non-exhaustive list):

  • EXMARaLDA Partitur Editor
  • TreeTagger
  • Arborator
  • ELAN
  • MMAX2
  • RST Tool
  • WebAnno
  • MS Excel
