Download ANNIS3

ANNIS is an open source project distributed under the Apache Public License, Version 2.0. The system requires a PostgreSQL installation to run, and the server version also requires a running web-server (e.g. Apache Tomcat or Jetty).

ANNIS3

Current ANNIS3 distribution: Version 3.2.2 (change log)

  • ANNIS Kickstarter: annis-kickstarter-3.2.2-distribution.zip (local installation for Windows, Linux and Mac)
  • ANNIS server:
    • annis-gui-3.2.2.war (server GUI distribution)
    • annis-service-3.2.2-distribution.tar.gz (server service distribution)

Older distributions:

  • Annis-3.1.8 - final version of the ANNIS 3.1.X series (change log)
    • annis-kickstarter-3.1.8-distribution.zip (local installation for Windows, Linux and Mac)
    • annis-gui-3.1.8 (server GUI distribution)
    • annis-service-3.1.8-distribution.tar.gz (server service distribution)
  • Annis-3.0.1
    • annis-kickstarter-3.0.1-distribution.zip (local installation for Windows, Linux and Mac)
    • annis-gui-3.0.1 (server GUI distribution)
    • annis-service-3.0.1-distribution.tar.gz (server service distribution)

For installation instructions see the ANNIS User Guide and our Documentation page. Source code (including the latest experimental developments) is available together with our bugtracker/feature requests ('issues') from github.com (http://github.com/korpling/ANNIS/)

If you're interested in working with ANNIS but have trouble setting everything up, please don't hesitate to contact us. If you're already using ANNIS and would like to stay up to date on releases and current development, you can also join the ANNIS mailing list (see contact page).
