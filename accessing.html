Accessing ANNIS

ANNIS is currently being developed within research project D1: "Linguistic Database for Information Structure: Annotation and Retrieval", LAUDATIOand Georgetown Corpus Linguistics. ANNIS is an open source project under the Apache License 2.0(though access to many of the corpora in the system is restricted by licenses) and is based on freely available open source technologies ( PostgreSQL DB, the VaadinJava framework and a browser such as Mozilla FireFox). The source code is available from the download page. Currently three versions can be accessed on-line ( for best results use Mozilla FireFox,Google Chrome or Internet Explorer above version 8- older versions of IE have some known issues):

  • A demo versionwith a small sample of freely available corpora in Berlin (also available from the Demo Corporapage)
  • The Georgetown University ANNISruns some freely available corpora
  • The SFB 632instance SFB researchers and other partners (login required)
  • The latest experimental developer's version

If you are interested in obtaining a login or joining the ANNIS user and developer mailing list, please contact us at . We also welcome cooperation partnersinterested in developing interesting corpora or expansions to the pluggable architecture, such as visualization modules and query builder modules.
