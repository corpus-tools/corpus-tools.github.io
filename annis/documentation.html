Documentation

General reference

Users new to ANNIS should begin by reading the User Guide. The online documentation complements the guide by giving up to date information and indepth reference on customizing settings for administrators as well as doxygen generated documentation for developers.

  • User Guide 3.3.5 (pdf)
  • Online documentation (for administrators and developers)

Specific topics

  • ANNIS3 – Multiple Segmentation Corpora Guide – in depth tutorial on corpora with multiple/conflicting tokenizations, subtoken segmentations and spoken corpora with multiple overlapping speakers.
  • HTML Visualization Guide – detailed instructions on creating you own custom annotation-triggered HTML visualizations with CSS, including example tutorials
